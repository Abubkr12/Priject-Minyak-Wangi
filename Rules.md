# Rules

- Selalu gunakan model Gemini Flash (hindari model Pro).
- Selalu gunakan Google Search Grounding untuk akurasi data fakta.
- Batasan RPM, TPM, dan RPD mengikuti sheet "AI Limitation.xlsx".
- Bypass limitasi API dengan rotasi keys seperti yang sudah diimplementasikan di `fix-perfumes.js`.
- Kerjakan dengan akurasi dan pastikan struktur data project dan rancangan tidak rusak.
- Jangan gunakan stock emoji dalam UI/UX, jaga estetika.
- Jangan gunakan default alert browser.
